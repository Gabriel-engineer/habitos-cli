# Habitos-Cli

📋 Checklist de hábitos diários para o terminal — feito com Python, pytest e GitHub Actions

🚀 **Deploy:** [https://replit.com/@gabriellgomes1/habitos-cli](https://replit.com/@gabriellgomes1/habitos-cli)

## Como usar
```bash
python SRC/Habitos/main.py adicionar "Beber água"
python SRC/Habitos/main.py listar
python SRC/Habitos/main.py concluir "Beber água"
python SRC/Habitos/main.py resumo
```
